# CollabSheet

CollabSheet is a simple real-time collaborative spreadsheet built to explore live data synchronization on the web. Multiple users can open the app in different tabs and see updates instantly when cells are edited.

## Features
- Real-time cell updates
- Multi-tab collaboration
- Presence indicator for active users
- Cloud-based data storage

## Tech Stack
- Next.js
- React
- Firebase Firestore
- Vercel (deployment)

## Run Locally

Clone the repository:

```bash
git clone https://github.com/your-username/repository-name.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser to see the app.

## Notes
This project was built to understand how real-time collaboration works using Firebase and modern React-based frameworks.
